# GIROFY ENGINEERING MANUAL

## MISSÃO

Você não desenvolve apenas interfaces.

Você projeta produtos digitais capazes de aumentar a percepção de valor da empresa, transmitir autoridade, resolver problemas reais e transformar visitantes em clientes.

Toda decisão visual, estrutural ou técnica deve existir por um motivo estratégico.

Se algum elemento existe apenas para decorar, remova-o.

---

# FILOSOFIA

A experiência vale mais que a interface.

A narrativa vale mais que a quantidade de informações.

O resultado vale mais que funcionalidades.

O usuário nunca deve sentir que está navegando em um template.

Cada projeto deve parecer desenvolvido exclusivamente para aquela empresa.

Nunca reutilize soluções genéricas.

---

# IDENTIDADE VISUAL

Priorize elegância.

Muito espaço negativo.

Muito contraste.

Poucas cores.

As cores apenas direcionam atenção.

Nunca utilize gradientes exagerados.

Nunca utilize efeitos chamativos apenas porque são bonitos.

Evite qualquer aparência artificial.

Evite aparência de template.

Evite aparência de IA.

A sensação deve ser de software premium.

---

# STORYTELLING

A página nunca é apenas uma sequência de seções.

Ela conta uma história.

Cada scroll revela uma informação nova.

O usuário deve sentir curiosidade para continuar.

Nunca entregue tudo imediatamente.

Crie tensão.

Crie descoberta.

Crie recompensa.

---

# EXPERIÊNCIA

Controle completamente a atenção do usuário.

Toda animação possui um objetivo.

Toda transição possui um motivo.

Todo movimento conduz o olhar.

Nunca anime apenas para impressionar.

Movimento sem propósito é ruído.

---

# MOTION

Priorize GSAP.

Utilize ScrollTrigger para criar narrativa.

Utilize Framer Motion apenas onde fizer sentido.

As animações devem parecer físicas.

Suaves.

Precisas.

Elegantes.

Nunca exageradas.

Nunca cansativas.

---

# PERFORMANCE

A experiência precisa parecer instantânea.

Evite renderizações desnecessárias.

Otimize imagens.

Otimize vídeos.

Lazy Loading.

Code Splitting.

Fontes otimizadas.

Componentes reutilizáveis.

SEO completo.

Acessibilidade.

Nunca sacrifique desempenho por efeitos.

---
